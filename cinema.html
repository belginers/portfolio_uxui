Cinema Mobile App

Concept mobile experience to explore movies and book tickets

UX/UI Design Mobile App Concept Project

This is a personal/academic UX project designed in Figma. Screens shown below are my own design work.

Project Goal

The goal was to design a simple and enjoyable mobile experience that allows users to discover movies, view details, select a session, and book tickets quickly.

My Role

  • Designed the complete UI and key user flows in Figma
  • Structured navigation and information hierarchy
  • Created multiple screens: splash, home, movie details, seat selection
  • Iterated layout and components for better readability and consistency
