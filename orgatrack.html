Project Overview

OrgaTrack is a role-based dashboard concept designed to organize daily operations in an industrial environment. It centralizes attendance validation, task tracking, employee assignment, and operational requests. I designed the interface for four user roles: Employee, Team Leader, Supervisor, and Admin.

The Challenge

The main challenge was to keep the experience simple while supporting different permissions and workflows. Each role needed access to specific actions (validate attendance, assign employees, create tasks, approve deletions), without overwhelming the interface.

My Role

  • Gathered requirements through client discussions
  • Designed information architecture and role-based navigation
  • Created UI screens in Figma with reusable components
  • Iterated designs based on feedback and constraints

User Roles

Employee

  • Validate attendance
  • View daily tasks
  • Track progress

Team Leader

  • Validate team attendance
  • Assign employees to positions
  • Review daily summary

Supervisor

  • Create daily schedules
  • Create tasks and manage requests
  • Monitor production and alerts

Admin

  • Global overview dashboard
  • Approve task deletion requests
  • Monitoring and reporting
